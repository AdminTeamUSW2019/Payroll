= DATABASE SETUP =

Database should have the following fields:

- VARCHAR NOT NULL forename
- VARCHAR NOT NULL surname
- VARCHAR NOT NULL email_address
- INT NOT NULL salery
- INT NOT NULL days_worked_this_month
- INT NOT NULL monthly_expenses (default = 0)
